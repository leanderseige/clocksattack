# Clocks Attack!!!

## Building the Game

Download and install

```git clone git@github.com:leanderseige/clocksattack.git```

```npm install```

Build Game:

```npx webpack --config=webpack.prod.js``` ...then serve folder ```dist/```.

Development:

```npx webpack serve --config webpack.dev.js```
